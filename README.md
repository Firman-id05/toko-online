API Toko Online - FastAPI & SQLite

Proyek ini adalah backend API untuk sistem Toko Online sederhana yang dibuat menggunakan FastAPI dan menggunakan SQLite sebagai database-nya. Proyek ini mendukung operasi CRUD produk serta beberapa fitur laporan transaksi.

Fitur Utama
CRUD Products: Tambah, lihat, ubah, dan hapus data produk.
Laporan Analisis: Laporan total belanja customer, produk terlaris, dan lainnya.
Dokumentasi Interaktif**: Auto-generated Swagger UI (`/docs`).

Cara Menjalankan Secara Lokal

Clone repository ini:
   bash
   git clone [https://github.com/Firman-id05/toko-online.git](https://github.com/Firman-id05/toko-online.git)
